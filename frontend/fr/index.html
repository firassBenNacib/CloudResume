À propos de moi

Bonjour ! Je m’appelle Firas, ingénieur Cloud & DevOps passionné par AWS et l’automatisation. Je conçois des pipelines CI/CD robustes, j’administre l’infrastructure avec Terraform et Ansible, et j’explore Kubernetes et le GitOps pour simplifier la vie. J’ai aussi travaillé avec Azure, mais AWS reste mon terrain de jeu favori.

Projets

KubleOps - Plateforme DevOps Cloud-Native

Mise en place d’une infrastructure prête pour la production avec Terraform et AWS EKS en GitOps complet. VPC hautement disponible sur deux zones de disponibilité avec NAT configurable, endpoints activables et bastion en option, plus un groupe de nœuds managé en autoscaling avec Karpenter et IRSA.

Intégration d’ArgoCD et Helm pour la livraison via AWS Load Balancer Controller. Activation d’ExternalDNS et de Fluent Bit avec des add-ons EKS gérés, supervision bootstrap via script EC2 et CI/CD CircleCI pour builds, scans, push d’images et synchronisations GitOps.

Outils : AWS (EKS, EC2, VPC, ALB, ECR, IAM, Route 53, ACM, CloudWatch), Terraform, Helm, ArgoCD, CircleCI, Karpenter, ExternalDNS, Fluent Bit, Prometheus, Grafana, Trivy

EcoAuto - Hébergement Web Économique & Scalable

Conception d’une infrastructure AWS hautement disponible et optimisée en coûts avec Terraform : EC2 Auto Scaling, RDS, site statique S3, CloudFront et Lambda@Edge pour la bascule d’origine. Automatisation serverless (arrêt/démarrage EC2/RDS, mises à jour DNS, synchro SG CloudFront).

Outils : AWS (EC2, ASG, RDS, S3, CloudFront, Lambda@Edge, Route 53, ACM, IAM, CloudWatch, EventBridge), Terraform, GitLab CI, Docker

Cloud Resume Challenge - Édition AWS

Réalisation complète du Cloud Resume Challenge avec AWS & Terraform. Archi serverless : site S3, API Gateway, Lambda, DynamoDB. CI/CD via GitHub Actions, supervision en temps réel via CloudWatch + SNS.

Outils : AWS (S3, CloudFront, ACM, Route 53, Lambda, API Gateway, DynamoDB, CloudWatch, SNS, IAM), Terraform, GitHub Actions

Plateforme d’Automatisation des Déploiements - Vermeg

Plateforme intelligente pour automatiser les déploiements sur WildFly & IBM WebSphere. Workflows rationalisés avec Ansible, CI/CD et scripts BDD. Déploiement sur OpenShift, tests automatisés multi-environnements.

Outils : Jenkins, Ansible, PostgreSQL, Oracle SQL, OpenShift, Selenium

OptiChat - Système de Messagerie Scalable

Développement d’un système de messagerie évolutif chez Vermeg, avec KEDA pour l’autoscaling et Argo CD pour des déploiements GitOps fluides. Supervision avec Prometheus & Grafana pour des perfs au top.

Outils : Docker, Kubernetes, Argo CD, Jenkins, Helm, KEDA, Prometheus, Grafana, ActiveMQ
