Projects

Cloud Resume Challenge – AWS Edition

Fully completed the Cloud Resume Challenge using AWS and Terraform. Designed a serverless infrastructure with an S3-hosted static website, API Gateway, Lambda, and DynamoDB. Implemented CI/CD with GitHub Actions and added real-time monitoring with CloudWatch + SNS.

Tools: AWS (S3, CloudFront, ACM, Route 53, Lambda, API Gateway, DynamoDB, CloudWatch, SNS, IAM), Terraform, GitHub Actions

EcoAuto – Cost-Effective Web Hosting Architecture

Designed and deployed a cost-optimized, highly available AWS infrastructure using Terraform. Integrated EC2 Auto Scaling, RDS, S3 static hosting, CloudFront, and Lambda@Edge for backend origin routing. Implemented full automation with serverless Lambda functions for scheduled EC2/RDS start-stop, backend origin switching, Route 53 DNS updates, and dynamic CloudFront security group synchronization.

This project serves as the base template powering infrastructure for projects like Tal9i7.tn.

Tools: AWS (EC2, ASG, RDS, S3, CloudFront, Lambda@Edge, Route 53, ACM, IAM, CloudWatch, EventBridge), Terraform, GitLab CI, Docker

Release Automation Platform – Vermeg

Built a smart platform to automate enterprise deployments on WildFly and IBM WebSphere. Streamlined release workflows with Ansible, CI/CD, and database scripting. The system was deployed on OpenShift and included automated tests with a multi-environment CI/CD pipeline for development, testing, and production.

Tools: Jenkins, Ansible, PostgreSQL, Oracle SQL, OpenShift, Selenium

OptiChat – Scalable Messaging System

Created a scalable messaging system at Vermeg, using KEDA for auto-scaling and Argo CD for smooth GitOps deployments. Monitoring with Prometheus and Grafana ensured peak performance.

This repository is a demonstration/prototype version of the real platform.

Tools: Docker, Kubernetes, Argo CD, Jenkins, Helm, KEDA, Prometheus, Grafana, ActiveMQ
