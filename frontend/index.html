About Me

Hey! I'm Firas, a Cloud & DevOps Engineer who loves AWS and tries to automate everything. I'm always building solid CI/CD pipelines, managing infrastructure with Terraform and Ansible, and exploring Kubernetes and GitOps to make life easier. I've also worked a bit with Azure, but AWS is where my heart is. I enjoy solving real problems and making technology work seamlessly.

Projects

KubleOps - Cloud-Native DevOps Platform

Set up a production-ready infrastructure using Terraform and AWS EKS with full GitOps deployment. Built a highly available VPC across two AZs with configurable NAT and toggleable endpoints, plus optional bastion access. Ran a managed node group with autoscaling, added Karpenter for right-sized capacity, and used IRSA for secure pod access.

Integrated ArgoCD and Helm for delivery through the AWS Load Balancer Controller. Enabled ExternalDNS and Fluent Bit with managed EKS add-ons. Bootstrapped monitoring with an EC2 script and wired CI/CD in CircleCI for builds, scans, image pushes, and GitOps syncs.

Tools: AWS (EKS, EC2, VPC, ALB, ECR, IAM, Route 53, ACM, CloudWatch), Terraform, Helm, ArgoCD, CircleCI, Karpenter, ExternalDNS, Fluent Bit, Prometheus, Grafana, Trivy

EcoAuto - Cost-Effective Web Hosting Architecture

Designed and deployed a cost-optimized, highly available AWS infrastructure using Terraform. Integrated EC2 Auto Scaling, RDS, S3 static hosting, CloudFront, and Lambda@Edge for backend origin routing. Automated EC2 and RDS start-stop, DNS updates, and CloudFront security group syncing with Lambda functions.

Tools: AWS (EC2, ASG, RDS, S3, CloudFront, Lambda@Edge, Route 53, ACM, IAM, CloudWatch, EventBridge), Terraform, GitLab CI, Docker

Cloud Resume Challenge - AWS Edition

Fully completed the Cloud Resume Challenge using AWS and Terraform. Designed a serverless infrastructure with an S3-hosted static website, API Gateway, Lambda, and DynamoDB. Implemented CI/CD with GitHub Actions and added real-time monitoring with CloudWatch + SNS.

Tools: AWS (S3, CloudFront, ACM, Route 53, Lambda, API Gateway, DynamoDB, CloudWatch, SNS, IAM), Terraform, GitHub Actions

Release Automation Platform - Vermeg

Built a smart platform to automate enterprise deployments on WildFly and IBM WebSphere. Streamlined release workflows with Ansible, CI/CD, and database scripting. The system was deployed on OpenShift and included automated tests with a multi-environment CI/CD pipeline.

Tools: Jenkins, Ansible, PostgreSQL, Oracle SQL, OpenShift, Selenium

OptiChat - Scalable Messaging System

Created a scalable messaging system at Vermeg using KEDA for auto-scaling and Argo CD for GitOps deployments. Monitored everything with Prometheus and Grafana to ensure peak performance.

This repository is a simplified prototype of the full system.

Tools: Docker, Kubernetes, Argo CD, Jenkins, Helm, KEDA, Prometheus, Grafana, ActiveMQ
